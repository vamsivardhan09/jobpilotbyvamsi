# 🚀 AI Career Assistant Platform  
### Smart Job Discovery • Resume Optimization • Voice-Based Mock Interviews  

---

## 📌 Overview
The AI Career Assistant Platform is a full-stack intelligent system designed to **help job seekers improve their chances of getting hired** through AI-driven resume analysis, job matching, and interview preparation.

It combines **Generative AI, real-time voice interaction, and job intelligence** to create a complete career support ecosystem.

---

## ❗ Real-World Problem
Job seekers today struggle with:

- ❌ Applying blindly without knowing job fit  
- ❌ Low ATS (Applicant Tracking System) match rates  
- ❌ Lack of personalized interview preparation  
- ❌ No real-time feedback on communication skills  
- ❌ Time wasted searching across multiple job portals  

👉 Result: Low interview calls and high rejection rates  

---

## 💡 Solution
This platform solves the above problems by providing:

- 📄 Intelligent resume analysis  
- 🎯 AI-powered job matching from multiple sources  
- ✍️ Resume optimization tailored to specific job descriptions  
- 🎙️ Real-time voice-based mock interviews  
- 🧠 Personalized interview question generation  

👉 Helping users move from **“random applications” → “strategic job targeting”**

---

---

## ⚙️ Key Features

### 🔍 1. AI Resume Analysis & Job Matching
- Upload resume → system extracts skills and experience  
- Fetches job listings from multiple sources  
- Ranks jobs based on **skill relevance and match score**

---

### ✍️ 2. Resume Tailoring (Generative AI)
- Rewrites resume for a specific job description  
- Improves keyword alignment for ATS systems  
- Enhances chances of shortlisting  

---

### 🎙️ 3. Voice-Based Mock Interview System
- Real-time AI interviewer using Web Speech API  
- Speech-to-text + text-to-speech interaction  
- Simulates real interview environment  

---

### 🧠 4. AI Interview Question Generator
- Generates technical & behavioral questions  
- Based on:
  - Resume content  
  - Skills  
  - Projects  
- Uses prompt engineering with OpenAI APIs  

---

## 🧰 Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Flask  
- **Database:** Supabase  
- **AI/ML:** OpenAI APIs (GPT-based prompt engineering)  
- **Voice System:** Web Speech API (Speech Recognition + TTS)
